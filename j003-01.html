Introduction to Control Flow

Programs don't always run in a straight line. Control statements act like traffic signs, telling the code where to go.

  • Selection (Decision) Choosing paths based on criteria (e.g., if-else).
  • Loops (Repetition) Repeating tasks until a goal is met.
  • Control Keywords Using 'break' or 'continue' to jump around.

The Mechanics of If-Else

Standard Syntax

if ( condition ) {
    // Executes if condition is TRUE
    System.out.println("Hello World");
} else {
    // Executes if condition is FALSE
    System.out.println("Goodbye World");
}
Critical Rule: An else cannot exist by itself. It must follow an if.

Logic Simulator

  result = "Pass";
} else {
  result = "Fail";
}

3

Evaluating Conditions

Valid Boolean Expressions

  • age >= 21 Relational
  • score == 100 Equality
  • (x > 0) && (x < 10) Logical

Beginner Pitfalls (Invalid)

  • age = 21 Assignment, not check!
  • num1 + num2 Returns number, not boolean
  • else (condition) Syntax error
4

Syntax Variations

1. The Omitted 'else'

Sometimes we only care if a condition is true. If false, we just continue with the rest of the code.

if (isHungry) {
    System.out.println("Eat snack");
}
// No else needed!
System.out.println("Go to sleep");

2. Omitting Braces { }

If you only have one statement, braces are optional (but recommended!).

if (score == 100)
    System.out.println("Perfect!");
else
    System.out.println("Try again");
5

The Shortcut: Ternary Operator

Want to write an if-else in just one line? Use the Ternary Operator. It's perfect for assigning values based on a condition.

Long Way (Standard)

String result;
if (age >= 18) {
    result = "Adult";
} else {
    result = "Minor";
}

Short Way (Ternary)

String result = (age >= 18) ? "Adult" : "Minor";

How to read it:

  1. Check the condition in ( )
  2. ? means "If True, use this"
  3. : means "Otherwise, use this"
6

Multi-Branching: if-else if

What if you have more than two choices? Use else if to check multiple conditions in a row. The program stops at the first true condition it finds.

if (score >= 90) {
    System.out.println("Grade: A");
} else if (score >= 80) {
    System.out.println("Grade: B");
} else if (score >= 70) {
    System.out.println("Grade: C");
} else {
    System.out.println("Grade: F");
}

"Think of it as a ladder. You step on each rung until you find one that holds you (is true), then you stop climbing."

7

Advanced Logic: Nesting

Complex decisions often require checking one thing after another. We do this by putting an if statement inside another if.

Nesting Example

if (isWeekend) {
    if (isSunny) {
        System.out.println("Go to beach");
    } else {
        System.out.println("Stay home");
    }
} else {
    System.out.println("Go to work");
}
