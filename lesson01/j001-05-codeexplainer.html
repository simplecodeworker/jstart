WelcomeInMessageDialogBox.java
import javax.swing.*; //or //import javax.swing.JOptionPane; //specific import
/* Example WelcomeInMessageDialogBox.java:    This application program displays Welcome to Java!    in a message dialog box. */
public class WelcomeInMessageDialogBox {     public static void main(String[] b) {     // Display Welcome to Java! in a message dialog box         JOptionPane.showMessageDialog(null, "Welcome ET0885-B...all of you!\nDon't be late");         JOptionPane.showMessageDialog(null, "Have you finished downloading IDE?");
        // below is another way to display message with more customized info         // JOptionPane.showMessageDialog(null, "Hello", "Welcome", JOptionPane.INFORMATION_MESSAGE);
        System.exit(0); /* needed to release resources when exiting GUI */     } }

Step 1: Importing Packages

Bringing in Tools

Before we can use JOptionPane, we need to tell Java where to find it.

  • import javax.swing.*: The asterisk (*) means "import everything in this toolbox".
  • It's common to import the whole library for GUI apps.
Step 2: Comments

Documentation

Text between /* ... */ is ignored by the computer. It is meant for humans to read so they understand the purpose of the program.

Step 3: Class Definition

The Program Container

Everything in Java lives inside a class.

The name WelcomeInMessageDialogBox must exactly match the filename.

Step 4: Main Method

The Starting Point

This is the "engine" of your program. When you hit run, Java looks for this exact line to begin execution.

Step 5: Dialog Box 1

Creating a Pop-up

JOptionPane.showMessageDialog is a built-in function that creates a window.

null: Standard window placement
"...": The text to display
\n: Creates a new line!
Step 6: Sequence

One after another

Java runs code line by line. The second dialog box will only appear after the user clicks "OK" on the first one.

Step 7: Customization

Advanced Options

You can customize the title bar and icon. INFORMATION_MESSAGE adds a little "i" icon to the window.

Step 8: System Exit

Cleaning Up

In GUI programming, the program might keep running in the background even if the window is closed.

System.exit(0) ensures the program stops completely and frees up your computer's memory.
