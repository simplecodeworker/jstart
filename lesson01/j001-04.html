Key Logic

  • • Modal: This dialog is "modal", meaning it blocks interaction with the parent until closed.
  • • Icon Behavior: The icon changes automatically based on the messageType integer.
  • • Return: This specific method returns void. Use showConfirmDialog if you need user input.
