1

Identify and correct the errors in the following codes:

public static int calculateSum(int num1, int num2) {
    int sum = num1 + num2;
}

Corrected Code

public static int calculateSum(int num1, int num2) {
    int sum = num1 + num2;
    return sum; // Added missing return statement
}

Error: The method is declared to return an int but lacks a return statement.

2

Identify and correct the errors in the following codes, hence, determine the result when parentMethod( ) is called.

public static void parentMethod() {
    double d = 0.1;
    System.out.println("The square of" + d + " is " + getSquare(d));

    public static double getSquare(double s) {
        return s * s;
    }
}

Correction & Result

Error: In Java, you cannot nest a static method inside another method. getSquare must be moved outside parentMethod.

Result: The square of 0.1 is 0.01

3

Identify and correct the errors in the following codes and hence, determine the result when the program is executed.

public class Diff {
    public static void main(String[] args) {
        double num1 = 6; 
        double num2 = 10.0; 
        System.out.println("The difference between " + num1 + " and " + num2 + " is " + calculateDiff(double num1, double num2));
    }

    public static double calculateDiff(double d1, double d2) {
        double diff = d1 - d2;
        return diff;
    }
}

Correction & Result

Error: When calling calculateDiff, you should not specify data types in the arguments. Remove double from the call.

Corrected Call: calculateDiff(num1, num2)

Result: The difference between 6.0 and 10.0 is -4.0

Identify and correct the errors in the following program and hence, determine the result when the program is executed.

public class Test {
    public static void main(String[] args) {
        method1(5);
        System.out.println(method2(-0.5, -5, 7.8));
    }  

    public static void method1(int n, int m) {
        n += m;
    }

    public static double method2(double n, float m, float x) {
        float temp = Math.ceil(n);
        if (temp > 0) return 1;
        else if (temp == 0) return Math.round(m);
        else if (temp < 0) return -1;
    }
}

Correction & Result

  • method1(5): Argument mismatch (needs two int).
  • Math.ceil(n): Returns double, cannot cast to float temp without explicit casting.
  • method2: Needs a default return statement at the end of the method body.

Result (if fixed): -5.0 (since ceil of -0.5 is 0.0, it returns round of -5)

5

What is pass by value? Show the result of the following method call:

public class Test {
    public static void main (String[] args) {
        int maxValue = 0;
        max (1, 2, maxValue);
        System.out.println("The maximun value is: " + maxValue);
    }

    public static int max (int value1, int value2, int maxValue) {
        if (value1 > value2)
            maxValue = value1;
        else
            maxValue = value2;
        return maxValue;
    }
}

How do you rectify this problem?

Analysis

Pass by Value: Java copies the value of the variable into the method parameter. Changing maxValue inside max() does not change the original variable in main().

Result: The maximum value is: 0

Rectification: Assign the returned value back to the variable:
maxValue = max(1, 2, maxValue);
