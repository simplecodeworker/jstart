The Math Table Program

Question: Write a program that prints a table showing the square root of numbers from 0 to 20 (incrementing by 2) using Math.sqrt().

Java 
public class SqrtTable {
    public static void main(String[] args) {
        // Print Header
        System.out.printf("%-10s %-15s%n", "Number", "Square Root");
        System.out.println("---------------------------");

        // Loop from 0 to 20, stepping by 2
        for (int i = 0; i <= 20; i += 2) {
            double result = Math.sqrt(i);
            
            // Format output to 4 decimal places
            System.out.printf("%-10d %-15.4f%n", i, result);
        }
    }
}

⚠️ Watch Out: printf vs println

Use printf for table alignment. %-10s creates a column 10 characters wide, left-aligned.

⚠️ Watch Out: Return Types

Even if you pass an int (like 4), sqrt() always returns a double (2.0).

Part (e)

Why must "Math." be added in front of method name?

Because sqrt() is a static method defined inside the Math class. Since we are not creating an object of the Math class (and it's a utility class), we must access the method using the class name.

Technical Concept: This is known as "Namespace" or "Class-level access". It prevents confusion if you wrote your own method named sqrt() elsewhere.
