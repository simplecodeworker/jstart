Question 1

1. What is the array index data type? What is the lowest index?

Answer:

The array index data type is int. The lowest index is 0.

Question 2

2. Which of the following statements are valid array declarations?

a. int i = new int(30);

b. double d[] = new double[30];

c. int i[]=(1, 4, 5, 9);

d. float f[] = {2.3, 4.5, 5.6}; (Missing 'f' suffix for literals)

e. double[] a = {2, 5.6, 7};

Answer:

b and e are valid.

  • a: Invalid syntax (uses parenthesis instead of brackets).
  • c: Invalid syntax (uses parenthesis for initialization).
  • d: Potentially invalid in Java (literal 2.3 is double; requires 2.3f for float).

Question 3

3. Indicate true or false for the following statements:

  • a. Every element in an array has the same data type.
  • b. The array size is fixed after the array is declared.
  • c. The array size is fixed after the array is created.
  • d. The elements in an array must be of primitive data type.

Answers:

a. True
b. False (can be initialized later)
c. True
d. False (can store objects)

Question 4

4.1 Declare an int array named: marks

4.2 Create and initialize the array content to: 100, 40, 50, 98

4.3 Assign the first element value to int variable mark4Alice

4.4 Assign the last element value to int variable mark4Danney

4.5 Use for loop and marks.length to calculate the average value of all the elements, then print out the average value.

4.6 Put the above codes into a class Marks, and run it.

Answer Implementation:

public class Marks {
    public static void main(String[] args) {
        // 4.1 & 4.2
        int[] marks = {100, 40, 50, 98};

        // 4.3
        int mark4Alice = marks[0];

        // 4.4
        int mark4Danney = marks[marks.length - 1];

        // 4.5
        double sum = 0;
        for (int i = 0; i < marks.length; i++) {
            sum += marks[i];
        }
        double average = sum / marks.length;
        System.out.println("Average: " + average);
    }
}

Question 5

5. Show the output of the following program:

public class PassArray{
    public static void main(String[] args){
        int number = 0;
        int[] numbers = new int[2];
        
        m(number, numbers);
        System.out.println(" number is" +number + "  and numbers[0] is "  + numbers[0]);
    }

    public static void m(int x, int[] y){
        x = 3;
        y[0] = x;
    }
}

Output:

number is 0 and numbers[0] is 3

Explanation: Primitive 'number' is passed by value (remains 0), while array 'numbers' is passed by reference (content changes).
