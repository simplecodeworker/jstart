Java Mastery: Arrays & Exceptions

Detailed breakdown of Question 5.3: Safe user input and array processing.

Problem Context

Assume a method exists public static double avgArry(double[] a) { ,, } that finds the average in an array of floating-point values.

a) Prompt user to enter the array size, with InputMismatchException handled, till the array size is a valid No.
b) Declare and create a double array with size decided by user in a)
c) Prompt user to enter the array element value one by one using loop with Exception handled
d) Call method avgArry(double[ ] a) to calculate the average of all the elements’ values of the above array , then print out the average value.

a

Prompt user for valid array size

"Prompt user to enter the array size, with InputMismatchException handled, till the array size is a valid No."

Scanner input = new Scanner(System.in);
int size = 0;
boolean isValid = false;

while (!isValid) {
    try {
        System.out.print("Enter array size: ");
        size = input.nextInt();
        if (size <= 0) throw new Exception();
        isValid = true;
    } catch (InputMismatchException e) {
        System.out.println("Error: Please enter an integer!");
        input.next(); // WATCH OUT! Clear the buffer
    } catch (Exception e) {
        System.out.println("Error: Size must be positive!");
    }
}

Beginner Alert: The Scanner Trap

When nextInt() fails due to a mismatch, the "bad" input stays in the buffer. If you don't call input.next() in the catch block, your loop will run infinitely!

b

Declare and create the array

"Declare and create a double array with size decided by user in a)"

double[] numbers = new double[size];

Note

In Java, array sizes are fixed once initialized. You must ensure size is a non-negative integer before this step.

c

Populate array with error handling

"Prompt user to enter the array element value one by one using loop with Exception handled"

for (int i = 0; i < numbers.length; i++) {
    boolean inputOk = false;
    while (!inputOk) {
        try {
            System.out.print("Enter value for element " + i + ": ");
            numbers[i] = input.nextDouble();
            inputOk = true;
        } catch (InputMismatchException e) {
            System.out.println("Invalid input. Please enter a double.");
            input.next(); // Clear buffer again!
        }
    }
}

Index Warning

Beginners often confuse the index i (0 to length-1) with the "count" (1 to length). Your loop should always start at 0.

d

Calculate and Print Average

"Call method avgArry(double[ ] a) to calculate the average... then print out the average value."

double result = avgArry(numbers);
System.out.println("The average is: " + result);

Method Signature Tip

When calling the method, you pass only the variable name numbers. Do not use brackets like avgArry(numbers[]) in the call!

Full Solution Implementation

ArrayAverage.java 
import java.util.*;

public class ArrayAverage {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int size = 0;
        
        // (a) Handle size input
        while (true) {
            try {
                System.out.print("Enter array size: ");
                size = sc.nextInt();
                if (size > 0) break;
                System.out.println("Size must be positive.");
            } catch (InputMismatchException e) {
                System.out.println("Invalid integer input!");
                sc.next();
            }
        }

        // (b) Create array
        double[] myArr = new double[size];

        // (c) Fill array
        for (int i = 0; i < size; i++) {
            while (true) {
                try {
                    System.out.print("Enter element " + (i+1) + ": ");
                    myArr[i] = sc.nextDouble();
                    break;
                } catch (InputMismatchException e) {
                    System.out.println("Input must be a decimal value.");
                    sc.next();
                }
            }
        }

        // (d) Call and Print
        double average = avgArry(myArr);
        System.out.println("Result: " + average);
    }

    // Mock method for demonstration
    public static double avgArry(double[] a) {
        double sum = 0;
        for (double val : a) sum += val;
        return sum / a.length;
    }
}
