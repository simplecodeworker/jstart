Module 4: Exception Handling

Exception Handling Assignment

In this assignment, you will learn how to handle user-driven errors using Java's robust Exception Handling framework. You'll ensure your program doesn't crash when someone types "hello" instead of a number.

The Question

"Write a Java program that asks the user to enter an integer. If the input is valid, echo the number. If it is invalid, inform the user without crashing the program."

Core Requirements

Success Scenario 
// Output
Enter an integer: 10
You entered: 10
Program finished.
Invalid Input Scenario 
// Output
Enter an integer: hello
Invalid input. Please enter an integer.
Program finished.

Pro Tips & Hints

  • Use Scanner and nextInt() to read user input.
  • When text is entered where a number is expected, Java throws InputMismatchException.
  • Don't forget the import statement at the top of your file!

The Syntax Toolbox

IMPORT
import java.util.InputMismatchException;
STRUCTURE
try { ... }
catch (InputMismatchException e) { ... }
finally { ... }
