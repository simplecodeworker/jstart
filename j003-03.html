1. The while Loop

"Check then Act" Logic

The while loop is the gatekeeper. It checks a condition before allowing any code inside to execute. If the condition starts false, the code never runs.

Flow & Logic:

  • Condition: A boolean expression (true/false).
  • Body: Runs repeatedly as long as condition stays true.
  • Zero-Execution Risk: If condition is false initially, the body is skipped.

Pro Tip: Curly braces {} are optional for single statements, but using them is best practice to avoid bugs during future edits.

WHILE_SYNTAX.JAVA
while (condition) {
    // loop body executes 
    // while condition is true
}

Logic Visualization

2. The do-while Loop

"Act then Check" Logic

Guaranteed Execution

Unlike the while loop, the do-while loop executes the body at least once, regardless of the condition, because the check happens at the bottom (Post-test Loop).

Practical Example: Summing User Input

A classic use case where you need to perform an action (ask for a number) before you can check if you should stop (is it zero?).

  1. 1 Read the number from user.
  2. 2 Add to sum.
  3. 3 Check if number is 0. If not, repeat.
DO_WHILE.JAVA
do {
    System.out.print("Enter number (0 to quit): ");
    number = input.nextInt();
    sum += number;
} while (number != 0);

3. The for Loop

The "Compact" Efficiency Loop

Use the for loop when you know exactly how many times to repeat. It bundles initialization, testing, and updating into a single line.

Feature While Loop For Loop
Best Used When... Number of iterations is unknown. Number of iterations is known.
Readability Spread out over multiple lines. Compact; logic is in one header.
Initialization Happens before the loop starts. Happens inside the loop header.

Logic Analysis

  • Initialization: Runs once at start (e.g. int i = 0).
  • Condition: Checked before every iteration.
  • Update: Runs after each loop body execution.
FOR_EXAMPLE.JAVA
for (int i = 0; i < 5; i++) {
    System.out.println((i + 1) + " Welcome!");
}

4. Nested Loops: "Loops Within Loops"

Managing Dimensions

A nested loop is simply a loop inside another. For every one iteration of the outer loop, the inner loop completes all of its iterations.

Common Use Case: Grids

Think of a multiplication table or a coordinate grid. The outer loop handles the rows, while the inner loop handles the columns.

MULTIPLICATION_TABLE.JAVA
for (int i = 1; i <= 3; i++) { // Rows
    for (int j = 1; j <= 3; j++) { // Cols
        System.out.print(i * j + " ");
    }
    System.out.println();
}

5. Controlling Flow: break vs continue

The "Emergency Exit" and the "Skip Button"

The break Statement

Immediately terminates the innermost loop and moves the program to the line after the loop body.

// If sum reaches 6, loop STOPS

if (sum >= 6) break;

The continue Statement

Immediately ends the current iteration and jumps back to the header to check the condition for the next round.

// If item is 2, line below is SKIPPED

if (item == 2) continue; sum = sum + item;

6. Modern Java: Enhanced For-Loop

Cleaner, Faster Iteration

Feature Standard for Loop Enhanced for Loop
Focus Uses an index (i) to access elements. Accesses the element itself directly.
Complexity More "boilerplate" code (i=0; i<len; i++). Much cleaner syntax.
Best For When you need the index or need to modify the array. When you just need to read every item in order.
Syntax Logic

Declare a temporary variable that represents the current item in the array as the loop progresses.

for (String name : names) {
  System.out.println(name);
}
"Instead of numbers[i], use int index directly."

Summary Quick-Reference

for loop

Use when the number of repetitions is known.

break

Exit the loop entirely right now.

continue

Stop this round and start the next one.

Enhanced for

Iterate through every item in a collection easily.

The Infinite Loop Warning

Regardless of the loop you choose, always ensure your condition will eventually become false. Otherwise, you create an Infinite Loop, which will cause the program to hang or crash.
