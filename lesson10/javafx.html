JavaFX for Beginners

Your step-by-step guide to building beautiful desktop applications with Java.

What is JavaFX?

JavaFX is a software platform for creating and delivering desktop applications, as well as rich internet applications (RIAs) that can run across a wide variety of devices. It is the modern replacement for Java Swing, offering a much more flexible, CSS-stylable UI and hardware-accelerated graphics.

JavaFX vs. Java Swing

Feature JavaFX (Modern) Java Swing (Legacy)
UI Architecture Separated via FXML & Scene Builder UI definitions tangled inside Java Code
Styling Standard CSS (Cascading Style Sheets) Complex Java "Look & Feel" Method calls
Graphics Hardware-accelerated (Prism), 3D support Software-rendered, strictly 2D
Media Built-in audio, video, and web-view support Requires clunky external libraries
Status Actively maintained and updated (Gluon HQ) Maintenance mode, no new features

How to Install JavaFX into Eclipse

  1. You must use the JavaFX version that is the same as your JDK version. For example, JavaFX 26 is not compatible with JDK 23. You can open up a command prompt and type javac -version
  2. Download the SDK: Go to the Gluon JavaFX website and download the JavaFX SDK for your operating system. Unzip it to a memorable location (e.g., C:\javafx-sdk).
  3. Create a User Library: In Eclipse, go to Window > Preferences > Java > Build Path > User Libraries. Click New, name it "JavaFX", and click Add External JARs. Select all the .jar files inside your unzipped JavaFX lib folder.
  4. Add to Project: Right-click your Eclipse project, select Build Path > Add Libraries > User Library, and check your new JavaFX library.
  5. Add VM Arguments (Crucial!): Right-click your main class, choose Run As > Run Configurations. Go to the Arguments tab. In the VM arguments box, paste the following (replace the path with your actual path): 
    --module-path "C:\path\to\javafx-sdk\lib" --add-modules javafx.controls,javafx.fxml

Concepts/Libraries

Before writing code, it's crucial to understand how JavaFX structures its user interface and the tools available to build it.

The Scene Graph

JavaFX UIs are constructed using a Scene Graph—a hierarchical tree data structure representing all visual elements.

A visual representation of the JavaFX Scene Graph hierarchy.

  • 1. Stage: The main window provided by the operating system.
  • 2. Scene: The canvas inside the window that holds the Scene Graph.
  • 3. Nodes: Every visual item in the Scene Graph is a Node.
    • Root Node: The top-level layout (e.g., a VBox or BorderPane).
    • Branch Nodes: Layouts that can have children (other layouts or controls).
    • Leaf Nodes: UI controls that cannot have children (e.g., Button, Label).

Essential Libraries

Core Application

  • javafx.application.Application - The base class for all JavaFX apps.
  • javafx.stage.Stage - The top-level container (the actual window).
  • javafx.scene.Scene - The container for all visual content.

Layouts (Panes)

  • javafx.scene.layout.VBox - Arranges children vertically.
  • javafx.scene.layout.HBox - Arranges children horizontally.
  • javafx.scene.layout.BorderPane - Top, bottom, left, right, center.
  • javafx.scene.layout.GridPane - Row and column grid layout.

UI Controls

  • javafx.scene.control.Button - Standard clickable button.
  • javafx.scene.control.Label - Non-editable text display.
  • javafx.scene.control.TextField - Single-line text input.
  • javafx.scene.control.PasswordField - Masked text input.
  • javafx.scene.paint.Color - Used for styling and backgrounds.

JavaFX Scene Builder

While you can build UIs purely with Java code (as shown in this guide), professional development often relies on a visual tool called JavaFX Scene Builder.

  • Visual Drag-and-Drop: Design your UI rapidly by dragging controls onto a canvas instead of writing boilerplate layout code.
  • FXML: Scene Builder generates an .fxml file. This is an XML-based markup language that defines your UI structure independently from your Java code.
  • Clean Architecture: It encourages the Model-View-Controller (MVC) pattern, allowing you to cleanly separate your visual design (FXML) from your Java business logic (Controllers).

Your First App: Hello World

Every JavaFX application must extend the Application class and override the start() method. Here is the simplest app you can make. You can copy the code or use the interactive simulator below to step through it! 

import javafx.application.Application;
import javafx.scene.Scene;
import javafx.scene.control.Label;
import javafx.scene.layout.StackPane;
import javafx.stage.Stage;

public class HelloWorldApp extends Application {

    @Override
    public void start(Stage primaryStage) {
        // 1. Create a control (Label)
        Label helloLabel = new Label("Hello, JavaFX World!");

        // 2. Create a layout and add the control
        StackPane root = new StackPane();
        root.getChildren().add(helloLabel);

        // 3. Create a Scene using the layout (Width: 300, Height: 250)
        Scene scene = new Scene(root, 300, 250);

        // 4. Configure the Stage (Window)
        primaryStage.setTitle("Hello World");
        primaryStage.setScene(scene);
        
        // 5. Display the window
        primaryStage.show();
    }

    public static void main(String[] args) {
        // Launch the JavaFX application
        launch(args);
    }
}

Building a User Login Page

A GridPane is perfect for aligning forms. This example uses labels, text fields, and an action event on a button to process the "login". Step through the interactive simulator below to see how it builds up! 

import javafx.application.Application;
import javafx.geometry.Insets;
import javafx.geometry.Pos;
import javafx.scene.Scene;
import javafx.scene.control.*;
import javafx.scene.layout.GridPane;
import javafx.stage.Stage;

public class LoginApp extends Application {

    @Override
    public void start(Stage primaryStage) {
        // 1. Create a GridPane for layout
        GridPane grid = new GridPane();
        grid.setAlignment(Pos.CENTER);
        grid.setHgap(10); // Horizontal gap between columns
        grid.setVgap(10); // Vertical gap between rows
        grid.setPadding(new Insets(25, 25, 25, 25));

        // 2. Create UI Controls
        Label titleLabel = new Label("Welcome, Please Login");
        // Col 0, Row 0, ColSpan 2, RowSpan 1
        grid.add(titleLabel, 0, 0, 2, 1); 

        Label userNameLabel = new Label("Username:");
        grid.add(userNameLabel, 0, 1); // Col 0, Row 1
        TextField userTextField = new TextField();
        grid.add(userTextField, 1, 1); // Col 1, Row 1

        Label passwordLabel = new Label("Password:");
        grid.add(passwordLabel, 0, 2); // Col 0, Row 2
        PasswordField passwordBox = new PasswordField();
        grid.add(passwordBox, 1, 2); // Col 1, Row 2

        Button loginButton = new Button("Sign In");
        grid.add(loginButton, 1, 3); // Col 1, Row 3

        Label messageLabel = new Label();
        grid.add(messageLabel, 1, 4);

        // 3. Add Event handling to the button
        loginButton.setOnAction(event -> {
            String user = userTextField.getText();
            // In a real app, you would verify against a database
            messageLabel.setText("Login attempted for: " + user);
            messageLabel.setStyle("-fx-text-fill: green;");
        });

        // 4. Setup Scene and Stage
        Scene scene = new Scene(grid, 350, 275);
        primaryStage.setTitle("JavaFX Login");
        primaryStage.setScene(scene);
        primaryStage.show();
    }

    public static void main(String[] args) {
        launch(args);
    }
}

Dynamic UI: Sidebar Application

This advanced beginner example uses a BorderPane. The left region holds a VBox acting as a sidebar menu. The center region changes its background color and text based on which menu button is clicked. Use the simulator to see the layout and events in action! 

import javafx.application.Application;
import javafx.geometry.Insets;
import javafx.scene.Scene;
import javafx.scene.control.Button;
import javafx.scene.control.Label;
import javafx.scene.layout.*;
import javafx.scene.paint.Color;
import javafx.scene.text.Font;
import javafx.stage.Stage;

public class SidebarApp extends Application {

    @Override
    public void start(Stage primaryStage) {
        // Main Layout
        BorderPane root = new BorderPane();

        // --- CENTER AREA ---
        StackPane centerArea = new StackPane();
        Label centerText = new Label("Welcome! Select a menu item.");
        centerText.setFont(new Font("Arial", 24));
        centerArea.getChildren().add(centerText);
        
        // Default center background (White)
        centerArea.setBackground(new Background(new BackgroundFill(Color.WHITE, CornerRadii.EMPTY, Insets.EMPTY)));
        
        root.setCenter(centerArea);

        // --- SIDEBAR (LEFT AREA) ---
        VBox sidebar = new VBox(15); // 15px vertical spacing
        sidebar.setPadding(new Insets(20));
        sidebar.setBackground(new Background(new BackgroundFill(Color.web("#333333"), CornerRadii.EMPTY, Insets.EMPTY)));
        sidebar.setPrefWidth(200);

        // Create sidebar buttons
        Button btnDashboard = createMenuButton("Dashboard");
        Button btnProfile = createMenuButton("Profile");
        Button btnSettings = createMenuButton("Settings");

        sidebar.getChildren().addAll(btnDashboard, btnProfile, btnSettings);
        root.setLeft(sidebar);

        // --- EVENT HANDLERS ---
        btnDashboard.setOnAction(e -> {
            centerText.setText("Dashboard View");
            // Change background to Light Blue
            centerArea.setBackground(new Background(new BackgroundFill(Color.LIGHTBLUE, CornerRadii.EMPTY, Insets.EMPTY)));
        });

        btnProfile.setOnAction(e -> {
            centerText.setText("User Profile View");
            // Change background to Light Green
            centerArea.setBackground(new Background(new BackgroundFill(Color.LIGHTGREEN, CornerRadii.EMPTY, Insets.EMPTY)));
        });

        btnSettings.setOnAction(e -> {
            centerText.setText("System Settings");
            // Change background to Light Coral
            centerArea.setBackground(new Background(new BackgroundFill(Color.LIGHTCORAL, CornerRadii.EMPTY, Insets.EMPTY)));
        });

        // --- SCENE SETUP ---
        Scene scene = new Scene(root, 700, 500);
        primaryStage.setTitle("JavaFX Dashboard Interface");
        primaryStage.setScene(scene);
        primaryStage.show();
    }

    // Helper method to style buttons uniformly
    private Button createMenuButton(String text) {
        Button btn = new Button(text);
        btn.setMaxWidth(Double.MAX_VALUE); // Make button fill VBox width
        btn.setStyle("-fx-background-color: #555555; -fx-text-fill: white; -fx-font-size: 14px; -fx-padding: 10px;");
        return btn;
    }

    public static void main(String[] args) {
        launch(args);
    }
}

Exploring Common UI Controls

JavaFX comes with a rich set of built-in controls. In this example, we'll build a simple Pizza Order form using a ComboBox (dropdown), RadioButtons, and CheckBoxes. Use the simulator to see how each control is initialized and added to the layout! 

import javafx.application.Application;
import javafx.geometry.Insets;
import javafx.scene.Scene;
import javafx.scene.control.*;
import javafx.scene.layout.VBox;
import javafx.stage.Stage;

public class ControlsApp extends Application {

    @Override
    public void start(Stage primaryStage) {
        // 1. Main Layout
        VBox root = new VBox(10);
        root.setPadding(new Insets(20));

        // 2. Title Label
        Label title = new Label("Pizza Order Form");
        title.setStyle("-fx-font-size: 18px; -fx-font-weight: bold;");

        // 3. ComboBox (Dropdown)
        Label sizeLabel = new Label("Select Size:");
        ComboBox sizeBox = new ComboBox<>();
        sizeBox.getItems().addAll("Small", "Medium", "Large");
        sizeBox.setValue("Medium");

        // 4. Radio Buttons (Exclusive choice)
        Label crustLabel = new Label("Crust Type:");
        ToggleGroup crustGroup = new ToggleGroup();
        RadioButton thinCrust = new RadioButton("Thin Crust");
        thinCrust.setToggleGroup(crustGroup);
        thinCrust.setSelected(true);
        RadioButton thickCrust = new RadioButton("Thick Crust");
        thickCrust.setToggleGroup(crustGroup);

        // 5. CheckBoxes (Multiple choice)
        CheckBox cheeseBox = new CheckBox("Extra Cheese");
        CheckBox spicyBox = new CheckBox("Spicy Sauce");

        // 6. Action Button & Output Label
        Button orderBtn = new Button("Place Order");
        Label summaryLabel = new Label();
        
        orderBtn.setOnAction(e -> {
            String crust = thinCrust.isSelected() ? "Thin" : "Thick";
            summaryLabel.setText("Ordered: " + sizeBox.getValue() + ", " + crust + " Crust");
            summaryLabel.setStyle("-fx-text-fill: #d97706; -fx-font-weight: bold;");
        });

        // 7. Add all controls to layout
        root.getChildren().addAll(
            title, sizeLabel, sizeBox, 
            crustLabel, thinCrust, thickCrust, 
            cheeseBox, spicyBox, orderBtn, summaryLabel
        );

        // 8. Setup Scene and Stage
        Scene scene = new Scene(root, 300, 400);
        primaryStage.setTitle("UI Controls Demo");
        primaryStage.setScene(scene);
        primaryStage.show();
    }

    public static void main(String[] args) {
        launch(args);
    }
}

Scene Builder & FXML

Writing UI layouts purely in Java can get tedious for complex screens. JavaFX Scene Builder is a visual drag-and-drop design tool that lets you design UIs without writing any code. It automatically generates FXML (an XML-based markup language) that your Java application can then load.

1. Installation & Setup

  1. Download: Go to the Gluon HQ website and download the Scene Builder installer for your operating system. Install it using the default settings.
  2. Link to Eclipse: Open Eclipse and navigate to Window > Preferences. Type "JavaFX" into the left-hand search box.
  3. Set Executable Path: Browse for the installed Scene Builder executable (e.g., C:\Users\YourName\AppData\Local\SceneBuilder\SceneBuilder.exe on Windows). Click Apply and Close.

2. Using Scene Builder

Creating the UI

  • In Eclipse, right-click your src folder > New > Other > JavaFX > New FXML Document. Name it Main.fxml.
  • Right-click Main.fxml in your package explorer and select Open with SceneBuilder.
  • In Scene Builder, drag a VBox from the Library panel (left) onto the Canvas (center).
  • Drag a Label and a Button into the VBox.
  • Save the file (Ctrl + S) and close Scene Builder. The FXML code in Eclipse will update automatically!
Visual Drag-and-Drop UI Design

3. Loading FXML in Java

Once your Main.fxml layout is ready, you need to load it into your Java application using the FXMLLoader class instead of building the Scene Graph manually. 

import javafx.application.Application;
import javafx.fxml.FXMLLoader;
import javafx.scene.Parent;
import javafx.scene.Scene;
import javafx.stage.Stage;

public class FxmlApp extends Application {

    @Override
    public void start(Stage primaryStage) throws Exception {
        // 1. Load the FXML file from the resources folder
        // Note: Main.fxml must be in the same package as this Java class
        Parent root = FXMLLoader.load(getClass().getResource("Main.fxml"));

        // 2. Attach the loaded UI to a Scene
        Scene scene = new Scene(root, 400, 300);
        
        // 3. Configure and show the Stage
        primaryStage.setTitle("App Loaded from FXML");
        primaryStage.setScene(scene);
        primaryStage.show();
    }

    public static void main(String[] args) {
        launch(args);
    }
}
