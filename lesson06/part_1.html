Understanding Parallel Arrays

What is it?

A Parallel Array is a structure where two or more arrays of the same size are used to store related data. Each index ($i$) acts as a "virtual link."

For example, names[0] and balances[0] belong to the same human, but Java treats them as totally separate lists.

Why Procedural?

In Procedural Programming, you only have basic data types (int, String, double). You don't have a "container" to hold a Name and a Balance together.

Because you can't create an Account type, you are forced to manage several "primitive" arrays and pray they stay synchronized.

// One array for names, one for money
String[] names = {"Jane", "John"};
double[] balances = {1000.0, 500.0};

// To find Jane's balance, you MUST know she is index 0
System.out.println(names[0] + ": " + balances[0]);
