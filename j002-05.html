Element Convention Rule Example
Class PascalCase Noun; start with Uppercase. UserAccount
Interface PascalCase Adjective/Noun; start with Uppercase. Serializable
Method camelCase Verb; start with lowercase. calculateTotal()
Variable camelCase Noun; start with lowercase. retryCount
Constant SNAKE_CASE Uppercase with underscores. MAX_WIDTH
Source File PascalCase Must match public class name. MyApp.java

PascalCase

The Mountain Range

High peaks at the start of every word. Every word stands tall like a mountain peak.

camelCase

The Dromedary Camel

Starts flat (lowercase) then humps up at the next word. Only one hump in the middle.

camelHump

SNAKE_CASE

The Hissing Snake

Long, flat shape punctuated by underscores, written in "LOUD" caps.
