Module 7: Subclasses

Reference: Q6.2.1 Solution

Parent Class

You previously created the Fan class. Use this completed code as a reference when building your subclass. Note that the properties radius and color are private and must be accessed via their getter methods.

Fan.java 
public class Fan {
  
  private double radius = 5;
  private String color = "white";

  public Fan() { }

  public double getRadius() {
    return radius;
  }

  public void setRadius(double dbRadius) {
    radius = dbRadius;
  }

  public String getColor() {
    return color; 
  }

  public void setColor(String strColor) {
    color = strColor; 
  }

  public String toString() {
    return "Fan [radius=" + radius + ", color=" + color + "]";
  }
}

Q7.1: Create BoxFan Subclass

Exercise

Create a subclass BoxFan of the Fan class. Your BoxFan must have:

  • Its own private property: int timing
  • 2 methods: setTiming(int time) and getTiming()
  • One method to override the parent's method public String toString() to print out all properties including timing.

Finally, write a Java Application class with a main method to create a BoxFan object, change its color, set its timing, and print its properties.

Workspace

Suggested Solution 

// 1. The BoxFan subclass
public class BoxFan extends Fan {
    
    // Its own private property
    private int timing;

    // Method to set timing value
    public void setTiming(int time) {
        timing = time;
    }

    // Method to get timing value
    public int getTiming() {
        return timing;
    }

    // Override parent's toString method
    @Override
    public String toString() {
        // Notice we must use getRadius() and getColor() because 
        // radius and color are private in the Fan class.
        return "BoxFan [radius=" + getRadius() + 
               ", color=" + getColor() + 
               ", timing=" + timing + "]";
    }
}

// 2. The Application class
class BoxFanApp {
    public static void main(String[] args) {
        BoxFan myBoxFan = new BoxFan();
        
        myBoxFan.setColor("Blue");
        myBoxFan.setTiming(120);
        
        System.out.println("Fan Details: " + myBoxFan.toString());
    }
}
