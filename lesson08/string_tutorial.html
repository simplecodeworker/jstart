6 String Part

1. Suppose that s1 and s2 are two strings. Which of the following statements of expressions are incorrect?

a. String s3 = s1 + s2;
b. String s3 = s1 - s2;
c. int i = s1.length( );
d. char c = s2(0);
Incorrect password.

Correct Answer

b and d are incorrect.

  • b: The subtraction operator (-) is undefined for Strings in Java.
  • d: The correct syntax to access a character is using the method s2.charAt(0), not parentheses.

2. Which of the following String declaration are correct?

a. String beta = new ("Hello World!");
b. String beta = new String("Hello World!");
c. String beta = "Hello World!";
Incorrect password.

Correct Answer

b and c are correct declarations.

  • a is invalid syntax (missing the class name after `new`).
  • b properly creates a new String object.
  • c uses a String literal, which is the standard and most efficient way to declare Strings.

3. Suppose there are following String objects:

String s1 = new String ("Hi");
String s2 = new String ("Hi");
String s3 = "Hi";
String s4 = "Hi";
String s5 = new String("hi");

Which of the following statement will print out "true"?

System.out.println(s1.equals(s2));
System.out.println(s1.equals(s3));
System.out.println(s1.equals(s5));
System.out.println(s1==s2);
System.out.println(s1==s3);
System.out.println(s3==s4);
System.out.println(s3.equals(s4));
System.out.println(s5.equalsIgnoreCase(s3));
Incorrect password.

Correct Answer

The following will print out true:

  • s1.equals(s2) (Contents match)
  • s1.equals(s3) (Contents match)
  • s3==s4 (String literal pool optimization, they point to the same memory reference)
  • s3.equals(s4) (Contents match)
  • s5.equalsIgnoreCase(s3) (Contents match ignoring case)

The == comparisons for `s1==s2` and `s1==s3` are false because `new` forces the creation of separate objects in memory.

4. Suppose that s1, s2, s3, s4 are four strings, given as follows:

String s1 = "Welcome to Java";

What are the results of the following expressions?

s1.charAt(0);
s1.indexOf('J');
s1.lastIndexOf("a");
s1.indexOf("j");
s1.substring(0, s1.indexOf("to"));
s1.startsWith("Wel");
s1.endsWith("va");
s1.toLowerCase( );
s1.length( );
Incorrect password.

Correct Answer

  • s1.charAt(0); ➔ 'W'
  • s1.indexOf('J'); ➔ 11
  • s1.lastIndexOf("a"); ➔ 14
  • s1.indexOf("j"); ➔ -1 (Case-sensitive, lower 'j' not found)
  • s1.substring(0, s1.indexOf("to")); ➔ "Welcome " (Includes space before 'to')
  • s1.startsWith("Wel"); ➔ true
  • s1.endsWith("va"); ➔ true
  • s1.toLowerCase(); ➔ "welcome to java"
  • s1.length(); ➔ 15

5. Examine the following declarations:

int area;
String name;

Which of the following is true?

a. area is a primitive variable, and name is a reference variable.
b. area is a reference variable, and name is a primitive variable.
c. both are primitive variables.
d. both are reference variables.
Incorrect password.

Correct Answer

a is true.

In Java, int is one of the 8 fundamental primitive data types. String is a class, which means any variable initialized with it becomes an object reference variable.
