Java Practical Q3.1 and Q3.2

Q3.1

Lucky Number Guessing

(a) Using If-Else Java 
import java.util.Scanner;

public class LuckyGuess {
    public static void main(String[] args) {
        Scanner input = new Scanner(System.in);
        int luckyNo = 7; // Hardcoded
        
        System.out.print("Guess my lucky number: ");
        int guess = input.nextInt();
        
        if (guess == luckyNo) {
            System.out.println("Congratulations!");
        } else {
            System.out.println("Thank You");
        }
    }
}

⚠️ Beginner Watch-out

Notice the use of == (double equals). Beginners often use a single =, which is for assignment, not comparison. This will cause a compilation error in Java inside an 'if' condition for integers.

(b) Using Switch Java 
switch (guess) {
    case 7:
        System.out.println("Congratulations!");
        break;
    default:
        System.out.println("Thank You");
}

⚠️ Beginner Watch-out

Never forget the break; statement! Without it, Java will "fall through" and execute the next case (the default), meaning even if you guess correctly, you'd see "Thank You" right after the congrats message.

(c) Ternary Operator Java 
String message = (guess == luckyNo) ? "Congratulations!" : "Thank You";
System.out.println(message);

⚠️ Beginner Watch-out

The ternary operator ? : is great for simple assignments, but don't over-nest them. If your logic gets complex, go back to if-else for readability.

Q3.2

Quiz Mark Grading

(a & b) If-Else-If with Validation Java 
if (mark < 0 || mark > 100) {
    System.out.println("Invalid mark! Must be 0-100.");
} else if (mark >= 80) {
    System.out.println("Grade: A");
} else if (mark >= 70) {
    System.out.println("Grade: B");
} else if (mark >= 60) {
    System.out.println("Grade: C");
} else if (mark >= 50) {
    System.out.println("Grade: D");
} else {
    System.out.println("Grade: F");
}

⚠️ Beginner Watch-out

Order matters! When using else if for ranges, start from the highest (80, 70, 60...) or the lowest. If you check mark >= 50 first, a score of 95 would match the first block and display "D".

(c1) Loop until -1 (While) Java 
System.out.print("Enter mark (-1 to quit): ");
int mark = input.nextInt();

while (mark != -1) {
    if (mark >= 0 && mark <= 100) {
        // ... Grade logic from above ...
    } else {
        System.out.println("Invalid mark!");
    }
    System.out.print("Enter mark (-1 to quit): ");
    mark = input.nextInt(); // Re-ask inside loop
}

⚠️ Beginner Watch-out

Infinite Loops: Beginners often forget to re-ask for input inside the while loop. If you don't update mark inside the brackets, the condition will never change, and the program will run forever!

(c2) Loop until -1 (Do-While) Java 
int mark;
do {
    System.out.print("Enter mark (-1 to quit): ");
    mark = input.nextInt();
    
    if (mark == -1) break; // Exit if -1
    
    if (mark >= 0 && mark <= 100) {
        // ... Grade logic ...
    } else {
        System.out.println("Invalid mark!");
    }
} while (mark != -1);

⚠️ Beginner Watch-out

The Semicolon: Unlike a standard while loop, a do-while loop must end with a semicolon after the while(...) condition.
