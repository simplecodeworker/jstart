Loops, Conditionals, and Formatting in Java

Part (a)

"Display all the odd numbers between 1 to 9—use if-else and for loop"

for (int i = 1; i <= 9; i++) {
    if (i % 2 ! = 0) { // Check if odd
        System.out.print(i + " ");
    } else {
        // Even numbers - do nothing or handle else
        // This is only for purpose of this assignment.
        // In actual implementation, you do not need to write an empty 'else' statement.
    }
}

Beginner Watch-outs:

  • The Modulo Operator (%): Beginners often forget that i % 2 returns the remainder. != 0 ensures we catch odd numbers.
  • Loop Boundaries: Using <= 9 includes 9. Using < 9 would exclude it.

Part (b)

"Modify your code to display all the odd number between 1-99 in 10 rows, with 5 numbers on each row. —use nested for loop"

int currentOdd = 1;
for (int row = 0; row < 10; row++) {
    for (int col = 0; col < 5; col++) {
        System.out.print(currentOdd + "\t");
        currentOdd += 2;
    }
    System.out.println(); // Move to next row
}

Beginner Watch-outs:

  • Println vs Print: Beginners often use println inside the inner loop, which puts every number on a new line. Use print for the row and println() only after the inner loop finishes.
  • Printf:You can also try out printf() - System.out.printf("%s\t", currentOdd);
  • Nested Logic: Note that the question specifically requests for the use of nested for loop. The outer loop represents the "vertical" movement (rows) and the inner loop represents the "horizontal" movement (columns).
  • Formatting: Using \t (tab) keeps the numbers aligned even when they change from 1-digit to 2-digits.

Part (c)

"Modify your code to display times table for even No. between 1 and 9, e.g. 2*2=4 ….8*8=64 in 4 rows and 4 columns."

// We target even numbers: 2, 4, 6, 8
for (int i = 2; i <= 8; i += 2) {
    for (int j = 2; j <= 8; j += 2) {
        System.out.print(i + "*" + j + "=" + (i * j) + "\t");
    }
    System.out.println();
}

Beginner Watch-outs:

  • Iteration Step: Note i += 2. This is more efficient than checking if (i % 2 == 0) inside a standard loop, though both work.
  • Order of Operations: When printing i * j, Java calculates the multiplication before concatenating with the string. However, wrapping math in parentheses (i * j) is a safe habit for beginners to ensure correct results.
  • The 4x4 Grid: Since there are exactly 4 even numbers (2,4,6,8) between 1 and 9, the nested loop naturally creates a 4x4 grid.

Part (c) Alternative

"Display times table for even No. between 1 and 9 by looping 1 to 9 and using the 'continue' keyword."

for (int i = 1; i <= 9; i++) {
    if (i % 2 != 0) continue; // Skip odd i

    for (int j = 1; j <= 9; j++) {
        if (j % 2 != 0) continue; // Skip odd j
        System.out.print(i + "*" + j + "=" + (i * j) + "\t");
    }
    System.out.println();
}

Beginner Watch-outs:

  • The 'continue' Keyword: It immediately jumps to the next iteration of the loop. Any code below it inside the loop block is ignored for that iteration.
  • Nesting 'continue': A `continue` inside the inner loop only affects that inner loop. It does not skip to the next row of the outer loop.
  • Logic Flow: Ensure the check happens before the print statement, or the odd number will be printed before the skip occurs!
