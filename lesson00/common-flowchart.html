1. The Basic Building Blocks

Every shape has a specific meaning. Using the right one is key to clarity.

START / END

Terminator

Used at the very beginning and the very end of every flowchart. It marks the entry and exit points.

PROCESS

Processing Block

Represents an action or a calculation. Examples: "Add 1 to Score" or "Bake the Cake for 20 mins".

DATA I/O

Input / Output

Used for getting data from a user or displaying results. Examples: "Get User Name" or "Print Message".

IS X > 10?

Decision

A diamond shape used for questions (Yes/No). It creates branches in your logic flow.

Flow Lines

Arrows that show the path of logic. They guide the reader from one step to the next.

The Loop

Loops aren't a single shape; they are formed when a flow line points back to an earlier step, usually after a Decision.
